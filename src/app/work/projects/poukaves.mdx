## Présentation

Poukaves est une plateforme anonyme et sécurisée permettant de signaler les problèmes sociaux, environnementaux et civiques qui affectent notre communauté. Le slogan : "Signalez pour un avenir meilleur".

## Fonctionnalités

- **Signalement anonyme** : les utilisateurs peuvent déposer des rapports sans révéler leur identité via un formulaire simplifié (description, catégorie, localisation)
- **Visualisation des données** : système d'analyse statistique avec graphiques interactifs présentant les tendances par catégorie et zones géographiques
- **Catégorisation** : fraude, corruption, pollution, infrastructure, incivilités et autres problématiques
- **Interface responsive** avec thème clair/sombre

## Architecture technique

- **Next.js** : framework React pour le rendu côté serveur et la génération statique
- **Prisma** : ORM pour la gestion de la base de données
- **Vercel** : déploiement et hébergement
